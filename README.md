# GMod Injector

Простой и удобный DLL-инжектор для **Garry's Mod** с современным ImGui-интерфейсом.

### Особенности
- Чистый и понятный интерфейс в тёмном стиле
- Автоматическое определение запущенного Garry's Mod (`gmod.exe` / `hl2.exe`)
- Удобный выбор DLL-файла через проводник
- Быстрый и стабильный инжект через `LoadLibrary`
- Красивая кастомная иконка
- Полностью на C++ + DirectX 11 + ImGui

### Скриншот
![GMod Injector](https://i.imgur.com/EuHdDSE.png)

### Как использовать
1. Скачай последнюю версию из Releases
2. Запусти `GMod Injector.exe` **от имени администратора**
3. Выбери нужный `.dll` файл
4. Нажми кнопку **INJECT**

Готово! DLL будет успешно загружена в Garry's Mod.

### Системные требования
- Windows 10 / 11 (x86)
- Garry's Mod (Steam x64 / x86 версия)
- Запуск от имени администратора

### Предупреждение
Данный инструмент предназначен только для образовательных целей и использования на локальных серверах.  
Использование на официальных серверах может привести к блокировке аккаунта.
